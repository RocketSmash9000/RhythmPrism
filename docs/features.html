Features

These are all the features RhythmPrism has to offer (so far so good).
Please note that this list may be updated as features are implemented.

Actual HD support

The moment you use a monitor with a resolution higher than 1080p Incredibox starts to have a ton of wasted  screen space. RP fixes this issue so that all screen is used screen.

Personalization

Every single texture you see, you can change it to a different one. Even the menu icons. There's nothing you can't change! (Except maybe some of the code)

Modularity

Each thing is independent of others. Delete one, add one. The code doesn't break, and it works just fine.

Easy to use

50% of the work is adding all the sprites and music. The other 50% is making the animations and changing a few things. But don't you worry, it's not as hard as it looks. If anything, it may take time.

Open source

With closed software you can't see what's inside it. You just have to trust it's not a virus. But RP is open source, so you can see what it does internally. And it comes with the bonus that you can alter the code to your liking!

Community support

If anything doesn't work, or if you're having trouble, you'll always find help. In the end, the least I want is you dropping a project due to technical difficulties.
